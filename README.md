# GraphQLとApollo

ブログ記事 [現代アプリケーションにおけるWeb APIの実装形式 (2) GraphQL](https://tknf.dev/application/modern-applications-web-api-graphql)のサンプルコードです。

## インストール
```sh
$ pnpm install
```

## セットアップ
GraphQLのスキーマから型定義ファイルを生成します。
```sh
$ pnpm run codegen
```

## サーバーの起動
```sh
$ pnpm run dev
```

※このプロジェクトは学習・検証用に作成されたものです。本番環境には使用しないでください。
